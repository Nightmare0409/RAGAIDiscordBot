RAG stands for Retrieval-Augmented Generation.
It improves AI responses by retrieving relevant documents before generating answers.

This project is a Discord bot that uses RAG to answer questions.
It retrieves information from a knowledge base and uses OpenAI to generate responses.

The bot is built using Python and discord.py.
It integrates with OpenAI API for generating responses.

RAG reduces hallucinations by grounding responses in real data.
